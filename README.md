What the Project Does:

The Discipline & Life Tracker is a web-based productivity application that helps users build habits and track daily progress.
Users can:
1. Create plans (e.g., Study Plan, Fitness Routine)
2. Add daily tasks with scoring (0–10 points)
3. Track daily performance
4. View progress history
5. Calculate final performance score (/100)
6. Write and manage personal notes
7. Sync data across devices using cloud storage
8. Customize UI themes
